# My2DGame

\#My2DGame

This is a clone of a 2D game based in Java



\##关于本项目



本项目完全参照自 RyiSnow 的教程《How to Make a 2D Game in Java》，是一个用于学习Java游戏开发基础知识以及\*\*面向对象程序设计\*\*的模仿练习作品。



* \*\*教程标题\*\*：How to Make a 2D Game in Java
* \*\*教程作者\*\*：RyiSnow
* \*\*教程平台\*\*：YouTube



\##项目目的：



* 学习Java游戏开发基础
* 实践面向对象编程思想（封装、基础、多态）
* 理解游戏主循环、碰撞检测、动画系统等核心概念
